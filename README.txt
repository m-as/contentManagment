Content Management Library

This JavaScript library contains a set of functions which can be used for adding all sorts of contents to your website.
The code has been written in pure JavaScript without the use of any JavaScript Library like jQuery.

The functions in the Content Management Library can be categorized into the following categories:
• Content Manipulation
• Adding New Content
• Changing Existing Content
• Removing Content
• Image Manipulation
• Link Manipulation
• Creating Banners
• Cookie Read/Write
